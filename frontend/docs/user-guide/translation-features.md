# Translation Features

The AI-Native Textbook platform offers comprehensive translation capabilities to make learning accessible in multiple languages.

## Available Languages

Currently, the platform supports translation in the following languages:

- English
- Urdu
- Spanish
- French
- German
- Chinese

## How to Use Translation

### Quick Translation Toggle

1. Look for the "Translate to Urdu" or "Translate to English" button at the top of each page
2. Click the button to quickly toggle between English and Urdu
3. The content will update immediately with the translated text

### Advanced Translation Settings

1. Click the "Show Options" button in the "Optional Features" section
2. Select your preferred translation language from the dropdown menu
3. Click "Apply Settings" to update the entire page
4. The translation will be saved in your browser for future visits

## Translation Quality

Our platform uses advanced AI models to provide high-quality translations. While translations are generally accurate, please note that technical terminology may occasionally vary between languages. If you notice any translation issues, please report them through our feedback system.

## Language Persistence

Your language preferences are saved in your browser's local storage, so they will persist between visits. You can reset to default settings using the "Reset to Default" button.

## Multi-Language Learning

You can switch between languages as needed to:
- Compare terminology between languages
- Enhance understanding of complex concepts
- Support multilingual learning environments

## Accessibility Considerations

Translation features are designed to be accessible to all users, including those using screen readers. Proper ARIA labels and semantic HTML ensure that translated content is properly announced by assistive technologies.
